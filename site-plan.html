Entertainment Web App

The mutlipurpose web application displays to users the latest and trending movies and series and makes them up to date with their various movies and series genre

Overview

Purpose

This website is intended to give user the ability to search for movies and series, bookmark them. Also give name of actors of the movies

Scenarios

What is the name of the main character in this movies?

What are the names of top 3 trending movies?
